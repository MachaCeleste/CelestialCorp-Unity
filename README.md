# Installation:
1. Copy all code from ``prebuilt/unity.src``
2. Paste all code into in game CodeEditor.exe
3. Compile to ``/bin/dddd``
4. Open a terminal and elevate to root (``sudo -s`` in any terminal window)
5. Run ``dddd``  (use ``unity`` for subsiquent launches), unity will rename itsself to ``unity``
6. Provide an IP for a "Hackshop" (preferably your own you get when you make an email address)
7. Unity will now auto setup making an appdata directory and downloading all needed libraries, it will also move libraries from ``/lib`` to the appdata directory
8. -h will display all commands in menu or fxplore
